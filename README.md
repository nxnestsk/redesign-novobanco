# Redesign App Bancária (REACT)

Um protótipo funcional e moderno de uma aplicação bancária desenvolvido em React, focado em UX/UI, animações fluidas e persistência de dados.

---

## Funcionalidades Principais

* **Navegação Completa:** Sistema de rotas dinâmicas com React Router.
* **Dashboard Financeiro:** Gráficos interativos com Recharts e resumo de movimentos.
* **Gestão de Cartões:** Simulação de bloqueio, visualização de PIN e ajuste de limites.
* **Poupanças Dinâmicas:** Criação de objetivos com armazenamento no LocalStorage.
* **Interface Responsiva:** Design adaptado para mobile e desktop usando Tailwind CSS.

---

## Tecnologias Utilizadas

* **React + Vite**
* **Tailwind CSS**
* **Framer Motion** (Animações)
* **Lucide React** (Ícones)
* **React Hot Toast** (Notificações)

---

## Como Correr o Projeto

1. Clone o repositório:
   `git clone https://github.com/nxnestsk/redesign-novobanco.git`
2. Instale as dependências:
   `npm install`
3. Inicie o servidor:
   `npm run dev`
